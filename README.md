# Enterprise HR + Payroll System

A Next.js corporate dashboard prototype for employee records, payroll, leave, attendance, recruitment, performance reviews, analytics, org charts, and audit logs.

## Run locally

```bash
npm install
npm run dev
```

Set `DATABASE_URL` for a PostgreSQL database when wiring the included schema to a real backend.

## Included

- Executive HR operations dashboard
- Salary, deductions, and payslip payroll table
- Leave approvals and attendance monitoring
- Recruitment, onboarding, performance review queues
- Analytics charts, organization chart, and audit log panels
- PostgreSQL schema starter in `database/schema.sql`
